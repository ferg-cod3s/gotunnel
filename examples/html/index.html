Example Commands

Here are some commands you can try with gotunnel:

Basic HTTP Tunnel (No Privileges)

# Start a tunnel for a local app on port 3000
gotunnel --proxy=builtin --no-privilege-check start \\
  --port 3000 --domain myapp --https=false

With Custom Proxy Ports

# Use custom proxy ports (useful in enterprise)
gotunnel --proxy=builtin --proxy-http-port 8080 \\
  --no-privilege-check start --port 3000 --domain myapp

Generate nginx Config

# Generate nginx configuration instead of running proxy
gotunnel --proxy=config start --port 3000 --domain myapp

Multiple Tunnels

# Terminal 1: Frontend
gotunnel --proxy=builtin start --port 3000 --domain frontend

# Terminal 2: API  
gotunnel --proxy=builtin start --port 8080 --domain api

# Terminal 3: Database Admin
gotunnel --proxy=builtin start --port 5432 --domain pgadmin
